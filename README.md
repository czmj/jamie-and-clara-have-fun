# hello-world

## Project setup
```
yarn install
yarn global add @vue/cli
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Run your end-to-end tests
```
yarn test:e2e
```
or
```
yarn test:e2e:headless
```

### Lints and fixes files
```
yarn lint
```
